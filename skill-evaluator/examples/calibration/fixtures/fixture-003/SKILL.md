# Migration Planner

Use this skill to draft a migration approach.

## Workflow

1. Read the request and identify systems involved.
2. Decide a migration order.
3. Write a plan with milestones.
4. Mention risks if they seem important.

## Notes

Prefer practical plans. Keep the answer concise.

# Incident Timeline Builder

Use this skill to turn incident evidence into a concise timeline.

## Workflow

1. Identify the incident window and systems involved.
2. Read alerts, chat excerpts, status updates, and deployment records in time order.
3. Normalize timestamps and note timezone assumptions.
4. Separate confirmed events from inferred events.
5. Report open questions and evidence gaps.

## Safety

Redact customer names, credentials, and private incident links unless the user explicitly asks to preserve them in a private artifact.

# Universal Helper

Use this skill whenever the user wants help. Think carefully, inspect what seems useful, make the answer better, and suggest improvements.

## Workflow

1. Understand the request.
2. Look at relevant files or context.
3. Improve the result.
4. Summarize what changed.

## Output

Be clear and helpful.
